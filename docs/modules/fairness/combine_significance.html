tost(low, upp, print_output=True, **kwargs)[source]¶

The two-one-sided t-test is a test of (non-)equivalence for two independent samples

TOST: two one-sided t tests

null hypothesis: m1 - m2 < low or m1 - m2 > upp alternative hypothesis: low < m1 - m2 < upp

where m1, m2 are the means, expected values of the two samples.

If the pvalue is smaller than a threshold, say 0.05, then we reject the hypothesis that the difference between the two samples is larger than the the thresholds given by low and upp.

Parameters

  • low (float) – Equivalence interval low < m1 - m2 < upp

  • upp (float) – Equivalence interval low < m1 - m2 < upp

  • print_output (boolean, optional) – Prints the dataframe results of the ANOVA analysis. Default is True

  • usevar (string, optional) – {“pooled”, “unequal”} If pooled, then the standard deviation of the samples is assumed to be the same If unequal, then Welsh ttest with Satterthwait degrees of freedom is used

  • weights (tuples of None or ndarrays, optional) – Case weights for the two samples. Default is None

  • transform (None or function, optional) –

    • If None (default), then the data is not transformed

    • If given a function, sample data and thresholds are transformed. If transform is log, then the equivalence interval is in ratio: low < m1 / m2 < upp

Returns

pandas.DataFrame with pvalue of equivalence test, test statistic and pvalue for lower/upper threshold test
