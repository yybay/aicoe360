valiant.fairness.statistical_significance¶

This module wraps statistical modules and automatically delivers the p-value, test statistic, with its associated model assumptions.

ANOVA (Analysis of Variance)¶

class valiant.fairness.statistical_significance.Anova(df, dep_var)[source]¶

Implementation of a suite of statistical models with automated checks for assumption violations

__init__(df, dep_var)[source]¶

Create an instance of the Anova

Parameters

  • df (pandas dataframe) – dataframe of dataset with independent and dependent variables

  • dep_var (string) – name of dependent variable column. All other variables in dataframe are assumed to be independent variables to be independent variables

anova(print_output=True, **kwargs)[source]¶

Runs the anova analysis, as well as checking for model assumption violations

Parameters

  • print_output (boolean, optional) – Prints the dataframe results of the ANOVA analysis. Default is True.

  • scale (float) – Estimate of variance, If None, will be estimated from the largest model. Default is None.

  • test (string) – {None, “F”, “Chisq”, “Cp”}. Test statistics to provide. Default is “F”.

  • ss_type (string or int) – {“I”, “II”, “III”} or {1, 2, 3}. The type of Anova test to perform. Default is 2

  • robust (string) – {None, “hc0”, “hc1”, “hc2”, “hc3”}. Use heteroscedasticity-corrected coefficient covariance matrix. If robust covariance is desired, it is recommended to use hc3.

Returns

Dataframe containing sum of squares, test statistic, and p-value
